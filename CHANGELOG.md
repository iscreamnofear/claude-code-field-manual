## [1.0.0] — 2026-02-23

First formally versioned release. Everything before this is pre-release history.

### Deliverables
- **PDF**: 92-page landscape edition (960x540, 16:9, ~19MB)
- **Landing page**: manual.ofirbloch.com — deployed via GitHub Pages
- **LinkedIn carousel**: 9-slide carousel (1080x1080) for launch post
- **Illustrations**: 12 book illustrations + 4 persona cards (marketer, ops, sales, revops)

### What's in the book
- 10 project playbooks (dashboard, landing page, email sequences, bot, etc.)
- 6 chapters: What is Claude Code, Coworking, Desktop Setup, Prompting, Safety, Troubleshooting
- Persona-driven framing (marketing, ops, sales, revops)
- Rebranded as "Ofir's Claude Code Field Manual"

### Infrastructure
- Custom domain: manual.ofirbloch.com (CNAME)
- Umami Analytics integrated
- GitHub Pages deployment from `docs/`
